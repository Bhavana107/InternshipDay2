# Titanic Dataset - Exploratory Data Analysis (EDA)

## Objective

To understand the Titanic dataset using descriptive statistics and data visualization techniques.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Plotly

## Tasks Performed

* Loaded and explored the dataset.
* Generated summary statistics (mean, median, standard deviation, etc.).
* Created histograms and boxplots for numerical features.
* Analyzed feature relationships using pairplots and a correlation matrix.
* Identified patterns, trends, and outliers through visualizations.
* Made feature-level inferences based on the analysis.

## Result

The exploratory data analysis provided insights into the dataset by revealing feature distributions, relationships, and important patterns, helping to better understand the data before applying machine learning models.
